Rocket Robot - Unreal Engine 5

Flappy bird remake in UE5. Written in C++ using assets from the Stack-O-Bot project.

Features

  • Infinite game loop (Until the player crashes into an object)
  • Player stays still while the environment elements move past
  • Random pillar hight each time the pillar loops back
  • Pillar movement is controlled with a single Actor Tick
  • High score is persistent between new games
  • UMG menus and HUDs
  • Each pillar is broken up into three sections and only the middle section size changes. This prevents warping of the bottom and top section.

Game will be available on Itch.io soon.

Jump Runes Game

The concept for this game is that you can move a character over the tiles and jump to reveal the rune below. The objective is to find all the rune matches in as little jumps as possible. There are 8 different runes each with their own colour. The game is build in Unreal 4.27.

Features

  • Character jump to show the rune below the tile
  • Runes location randomise on each run
  • High Score and moves tracker
  • Add custom assets
