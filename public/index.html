Christmas Gift Rush (November 2020)

A Christmas-themed slot game with animated snow and gifts.

Work Done

  • Matched Pixi.js text to translated languages with Spine Runtime text and animated the text
  • Implemented the UI provided by the design team
  • Implemented animation of background snow particles using Pixi.js particles

Techno Tumble (May 2020)

This was the first game we made that took out the conventional reel system and used a physics engine to simulate the dropping of the ball like a pin ball machine. Balls drop in from the top; depending on their colour, it will reward a win and the winning ball (symbols) will pop away!

Work Done

  • Created a tool to build our first physics game harness
  • Added the boundary walls to the harness
  • Built a custom tool to do polygon conversion between points and (x; y) coordinates, greatly decreasing the time it took to do minor changes to our physics harness

Loony Blox (January 2020)

This game had three different characters that drive around a map. Players win a “Feature” when they land on specific spots on the map.

Work Done

  • Created dust wind to cover the map
  • Implemented ‘Wild’ animation
  • Caused a random character to play their win animation during a win
  • Animated the smoke particles from the back of the car using Pixi.js particles

Calaveras Explosivas (2021)

This will be our second physics-based slot game. Once again, we have removed the standard reels from the game and added an effect to the skull that explodes upon win.

Work Done

  • Coded the skulls’ eyes to track symbols that are going explode
  • Implemented Spine Runtime symbols animations provided by the design team
  • Built the boundary wall that restricts and determines where the symbols will move and collect

This game is still in developement and will be release in 2021
